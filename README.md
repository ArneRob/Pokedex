# Pokedex 2.0

A browser-based Pokédex that fetches live data from the [PokéAPI](https://pokeapi.co/).

## Features

- Browse Pokémon with lazy loading
- Search by name or ID
- Detailed overlay with stats, types, and evolution chain
- Type-based color coding with icons

## Tech Stack

- Vanilla JavaScript, HTML, CSS
- PokéAPI (REST)

## Usage

Open `index.html` in a browser — no build step or server required.
